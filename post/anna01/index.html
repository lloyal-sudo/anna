SSH命令连接服务器

  • 即使用ssh命令打开另一Linux系统伪终端，从而进行远程操作。

SSH–>secure shell(安全外壳协议)，可简单理解为一种加密方式。

1.基础连接 

ssh 远程用户名@远程IP<br>
#例如：ssh kali@127.0.0.1

2.指定端口连接 

ssh -p 端口号 远程用户名@远程IP  #如果ssh默认端口号为22不加-p

3.退出连接 

exit
