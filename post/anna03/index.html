异步请求

  • 允许浏览器在向服务器发送请求后，不等待服务器响应完成便继续执行后续的代码，从而提高了Web应用的响应性和用户体验。
  • 主要通过JavaScript实现，典型就是Ajax
  • 相应内容一般是json数据，可更加快捷的定位请求

json 模块

  • 用于处理json数据格式的标准库模块，主要作用有：
    • 序列化（编码）
    # 将python对象（如字典、列表）转换为json格式的字符串,如：
#json.dumps({'name': 'anna', 'age': '21'})
import json

data = {"name": "anna", "age": "21"}
res = json.dumps(data)
print(res)
print(type(res))
# >> {"name": "anna", "age": "21"}
# >> <class 'str'>
    • 反序列化（解码）
    # 将json字符串解析为python对象，如：
# json.loads('{"name": "anna", "age": "21"}')
import json

data = '{"name": "anna", "age": "21"}'
result = json.loads(data)
print(result)
print(type(result))
# >> {'name': 'anna', 'age': '21'}
# >> <class 'dict'>
    • 文件读写
    # json.dump()：将python对象写入json文件。
# json.load()：从json文件读取并转换为python对象。
import json

# 读取：
# with open("file.json","r",encoding="utf-8") as f:
#     res = json.load(f)
#     print(res) # >>{'name': 'anna', 'age': '21'}
#     print(type(res)) # >> <class 'dict'>

# 写入
with open("file.json","w",encoding="utf-8") as f:
    loyal = {"name": "张三", "age": "18"}
    res = json.dump(loyal, f, indent=4, ensure_ascii=False) # 不进行编码
# file.json中写入了新的数据：
{
"name": "张三",
"age": "18"
}
